#### 四舍五入

```python
e, f, g, h = -0.35, -0.351, -0.45, -0.451

print("round() 函数（银行家舍入）:")
print(round(e, 1))  # -0.4（向最近的偶数舍入）
print(round(f, 1))  # -0.4
print(round(g, 1))  # -0.4（向最近的偶数舍入）
print(round(h, 1))  # -0.5

print("\n格式化输出（四舍五入到远离零）:")
print(f"{e:.1f}")  # -0.3
print(f"{f:.1f}")  # -0.4  
print(f"{g:.1f}")  # -0.5
print(f"{h:.1f}")  # -0.5
```
